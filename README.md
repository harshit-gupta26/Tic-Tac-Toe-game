## Usage
Open the TIc-Tac-Toe.html file in your browser (double-click or open via browser).

The game interface will load.

Players take turns clicking on empty cells to place their marks (X or O).

The game will announce a winner or a draw.

Click the Reset or Restart button to start a new game.

---

## How It Works

(1). The game board is a 3Ã—3 grid (9 cells).

(2). Players alternate placing their marks (X or O) by clicking on empty cells.

(3). After each move, the game logic checks for:

A winning line (row, column, or diagonal), or

A draw (if all cells are filled and no winner).

(4). If a win or draw is detected, the game stops accepting moves and shows the result.

(5). The player can reset to play again.

The core logic resides in a JavaScript file (e.g. script.js) which manages the game state, turn switching, checking win conditions, and UI updates.
